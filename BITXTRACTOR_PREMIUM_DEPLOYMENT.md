## üìà Usage Analytics

Track downloads with queries:
```sql
-- Total downloads
SELECT COUNT(*) FROM downloads;

-- Downloads by user
SELECT u.username, COUNT(d.id) as download_count
FROM users u
LEFT JOIN downloads d ON u.id = d.user_id
GROUP BY u.username
ORDER BY download_count DESC;

-- Recent downloads
SELECT u.username, d.file_name, d.download_timestamp
FROM downloads d
JOIN users u ON d.user_id = u.id
ORDER BY d.download_timestamp DESC
LIMIT 10;

-- Premium user statistics
SELECT subscription_tier, COUNT(*) as user_count
FROM users
GROUP BY subscription_tier;
```
